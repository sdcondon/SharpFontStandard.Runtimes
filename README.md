# SharpFontStandard.Runtimes

Runtime dependencies for SharpFontStandard. That is, the FreeType binaries structured in accordance with [package structure rules for architecture-specific files](https://docs.microsoft.com/en-us/nuget/create-packages/supporting-multiple-target-frameworks#architecture-specific-folders).

* `SharpFontStandard.Runtimes.WindowsOnly`: contains 32- and 64-bit Windows DLLs

## Usage

Just add the runtime package as a dependency along with SharpFontStandard.
